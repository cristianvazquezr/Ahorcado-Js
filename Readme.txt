La idea es crear el juego del ahorcado por que me premitia usar todas las cosas aprendidas hasta la clase 4. Para iniciar el juego se debe dar click en el botón "comenzar juego". Cuando se da click el programa elije una palabra aleatoria, luego te empieza a pedir que ingrese letras. El programa se encarga de corroborar que el usuario elija solo letras, y que elija de a una. Una vez que se ingresa la letra, se va guardando en una lista y se va completando la palabra secreta a medida que va adivinando. El programa corre dentro de un while donde valida la cantidad de vidas y si completó la palabra. Si alguna estas dos condiciones se cumple el programa finaliza.
